bố cục trang web có 7 phần cơ bản

  1. Logo: chứa logo trang web
  2. Menu: chứa menu chính điều hướng trang web
  3. Header: chứa nội dung phần trên cùng
  4. Left sidebar: chứa dữ liệu bên trái trang web
  5. Content: chứa nội dung trang web
  6. Right sidebar: chứa nội dung bên phải trang web
  7. footer: phần nằm dưới cùng trang web
