dùng css trong html

có 3 cách để dử dụng css trong html

  • cách 1:

    inline (ta đặt thuộc tính css vào thẻ mở html cần style)

    • exam:

    sử dụng css bằng inline

  • cách 2:

    internal (ta đặt các thuộc tính định dang bên trong cặp thẻ head)

    • exam:

    sử dụng css bằng internal

    sử dụng css bằng internal

  • cách 3:

    external (ta đặt các thuộc tính định dạng vào trong tập tin css)

    • exam:

    sử dụng css bằng external
